# react-redux-saga-typescript-boilerplate

A boilerplate for React, Redux and Typescript

### Environment configurations
#### System requirements
* node.js >= v12
* npm >= v6.7


#### Scripts

``` bash
# install dependencies
$ npm install

# run dev server
$ npm run dev

# check eslint
$ npm run lint

# build app for production
$ npm run build
```
